## 🌐 Option A — Upload to Wix Studio (Recommended)

1. Open Wix Studio
2. For each page, add an **HTML Embed** element (full width/height)
3. Copy-paste the contents of each HTML file into the embed
4. For the shared CSS: paste `style.css` contents into **Global CSS** in Wix Studio
5. Add your images to the 📸 placeholder spots
6. Embed YouTube videos into the ▶️ placeholder spots
7. Publish and copy your site link

---

## 💻 Option B — Open Directly in Browser (for previewing)

1. Open the `talking-together` folder
2. Double-click `index.html` to open the homepage
3. Navigate between pages using the menu

> **Note:** All links between pages work when opened from the folder directly.
